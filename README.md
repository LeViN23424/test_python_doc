Как выполнить проверки в терминале:

1. Клонируйте репозиторий и перейдите в папку проекта.
2. Установите зависимости (если требуется) .
3. Запустите модуль: python task_manager.py (или как называется ваш файл).
4. Последовательно выполните шаги из чек-листа и тест-кейсов, сверяя фактический результат с ожидаемым.
5. Для проверки сохранения/загрузки используйте команды save и load, а также перезапускайте программу между шагами.
